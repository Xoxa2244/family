# Family Task Tracker

Домашний трекер задач для семьи - веб-приложение для отслеживания выполнения дел членами семьи.

## Технологии

- Next.js 14 (App Router)
- TypeScript
- React
- dayjs для работы с датами
- localStorage для хранения данных

## Установка и запуск

```bash
# Установка зависимостей
npm install

# Запуск в режиме разработки
npm run dev

# Сборка для продакшена
npm run build

# Запуск продакшен версии
npm start
```

Приложение будет доступно по адресу [http://localhost:3000](http://localhost:3000)

## Функционал

### Страницы

1. **/login** - Страница входа
   - Логин по имени пользователя
   - Пароль для всех: `1234`

2. **/today** - Мои дела сегодня
   - Просмотр плана на сегодня
   - Добавление дел из доступных шаблонов
   - Отметка выполненных дел
   - Перенос дел на завтра

3. **/calendar** - Календарь-репортинг
   - Просмотр календаря по месяцам
   - Выбор пользователя для просмотра
   - Отображение выполненных/невыполненных дел
   - Статистика переносов

4. **/stats** - Сводная статистика
   - Статистика выполнения дел по пользователям
   - Определение чемпиона и аутсайдера месяца
   - Процент выполнения плана

5. **/admin** - Админ-кабинет
   - Управление справочником дел
   - Настройка плана по дням недели для каждого пользователя
   - Доступ: пароль `1234` (или автоматически для родителя)

## Структура данных

Все данные хранятся в localStorage под ключом `home-tracker-state`.
