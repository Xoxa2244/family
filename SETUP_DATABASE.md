# Настройка базы данных Supabase

## Шаг 1: Создание таблиц

1. Откройте Supabase Dashboard: https://supabase.com/dashboard
2. Выберите ваш проект
3. Перейдите в раздел **SQL Editor**
4. Скопируйте содержимое файла `supabase-schema.sql`
5. Вставьте SQL в редактор и выполните (Run)

Это создаст все необходимые таблицы:
- `users` - пользователи
- `task_templates` - шаблоны дел
- `task_template_assignments` - назначения дел пользователям
- `daily_quotas` - квоты по дням недели
- `task_instances` - конкретные инстансы дел
- `app_state` - текущее состояние приложения

## Шаг 2: Проверка подключения

Приложение автоматически инициализирует данные по умолчанию при первом запуске, если таблицы пустые.

## Настройки безопасности

В схеме включены политики Row Level Security (RLS), которые разрешают все операции для анонимных пользователей. Это подходит для семейного приложения, но если нужна дополнительная безопасность, можно настроить более строгие политики.

## Данные по умолчанию

При первом запуске приложение создаст:
- 4 пользователя: Родион (родитель), Нани, Роман, Ролан (дети)
- 4 шаблона дел: Английский, Физкультура, Чтение, Математика
- Квоты по дням недели для детей
